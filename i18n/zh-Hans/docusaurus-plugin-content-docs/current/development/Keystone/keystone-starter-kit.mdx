## 快速开始

### 1. 下载入门套件

`git clone https://github.com/curio-research/keystone-starter-kit.git`

### 2. 运行服务端

```bash
cd server
go mod tidy
go run main.go
```

### 3. 运行客户端

```bash
cd client
yarn
yarn start
```
