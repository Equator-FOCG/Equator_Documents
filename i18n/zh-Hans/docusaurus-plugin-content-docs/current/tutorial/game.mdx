## 游戏的看法

🕹️ **`游戏`**: 跳出现实的**实践**。

🎲 **`游戏`**不仅是创作和表达的媒介、社会交往的空间，更是**科技应用和创新的前沿**。

⛓️ **`游戏`**将有助于我们理解**区块链技术将为世界带来什么**。作为未来数字世界的缩影，他们充当了实验和创新的沙盒。虚构的世界具有真实的意义。也就是通过游戏的探索和需求改造现实！

## 可证明的游戏

多亏了零知识证明的魔力，我们可以通过验证链下创建的零知识证明来确保游戏的公平性。

但这意味着什么呢？考虑一盘国际象棋。我们的目标是让玩家相互信任对方的动作。

以一种直截了当的方法——考虑到国际象棋的简单规则——如果这是在区块链环境中，每一步都将是区块链上的交易。这是昂贵的。我们只想知道赢家，而不是每一步的具体走法。通过零知识证明和客户端通信，玩家可以建立一个状态通道，在链下共享动作并确保其有效性。

最后，可以向区块链提交零知识证明以确认游戏的公平性。这构成了一个可证明的游戏。

## 区块链游戏发展历史

### 早期的区块链游戏

`CryptoKitties`是一款区块链游戏，该游戏允许玩家在以太坊上购买、销售和创造非同质化代币（NFT），这些NFT代表虚拟猫。该游戏于2017年12月大受欢迎，导致以太坊网络拥堵，交易数量达到历史最高纪录，速度显著下降。

### GameFi 游戏

GameFi是"Game"和"Finance"组合而成的词。这些游戏主要围绕在链上托管资产和链下运行游戏。

`Axie Infinity`是一款挖矿区块链游戏，用户可以繁殖和战斗他们的NFT角色。它在2021年成为了超级热门游戏，全球的人们通过玩这款游戏可以获得可观的收入。
