# 自主世界 🗺️

全链游戏的终极目标“自主世界”，是对数字世界终极形态又一次阐述（上一次阐述是被过度营销化后烂大街的“元宇宙”）。自主世界作为人类对美好未来的共同想象，具有很好的号召力，而全链游戏作为实现该目标的重要途径，也被寄予厚望。

## 什么是世界？

一个世界是一个包含实体与足够一致的内部规则的容器。当一个包含了实体与规则的系统运转起来时，他就成为了一个世界。

## 什么是自主世界？

自治世界（AWs）是数字原生世界，由其智能合约定义和管理，完全存在于链上，并继承了区块链的独特功能：可组合、可扩展、无需许可和持久性。

它能够令人信服地致力于未来的状态：它将按照其规则存在，并且只要它愿意，它就会存在。

:::tip

至少可以说, 创建独立存在的虚拟世界的能力令人着迷。

:::
