# 全链游戏 🎯

## 什么是全链游戏？

链上游戏，也叫完全链上游戏、全链游戏。

一个游戏，**只有当它全部的游戏逻辑和数据都储存在链上时**，才能被称为是“全链游戏”。

:::备注

为什么要费心构建全链上游戏，并将状态和逻辑都放在链上，还要处理随之而来的所有复杂性。原因很简单 😊，就是要获得传统游戏基础架构所没有的特性。

:::

## 全链游戏的特点

### 1. 无许可的可组合性

`游戏新范式`：这意味着所有的玩家和开发人员都可以从高级客户端开始自定义游戏中的任何组件，包括引入插件或 mod。甚至是游戏的底层逻辑，都可以通过智能合约来扩展游戏的玩法。本质上，这是一种新的模式，在游戏中，每个游戏默认都是一个新的平台，而且无论你的团队多么小，你都可以成为内容巨头。

`参与度和社区治理`：全链游戏鼓励玩家积极参与游戏的运营决策和社区治理。玩家可以通过持有代币、投票等方式参与游戏的发展方向和决策过程，增加了玩家的参与度和归属感。

`游戏的开发遵循开放生态原则`：游戏合约和可访问的游戏客户端都是开源的。第三方开发者可以通过插件、第三方客户端、互操作智能合约，实现完全重新部署，定制甚至分叉自己的游戏体验。这反过来又使游戏开发者能够利用整个 (激励一致)社区的创造性产出；

### 2. 链上游戏的持久性

`真实性`：逻辑和规则通过智能合约实现，例如，游戏中的战斗，而不仅仅是所有权，都是在链上进行的；由于游戏逻辑和数据都存储在区块链上, 游戏规则和状态对所有人都是透明的。这使得玩家能够验证游戏的公平性和结果的正确性, 增加了游戏的可信度。

`一次上链, 永久运行`：即使游戏的原始开发者不再支持游戏, 只要这条链不停止出块, 游戏依然可以继续运行和发展。因为没有服务器可以被审查或关闭，这些体验实际上引入了一个全新的意义。不过这样一来，永恒的游戏将可能无法解决，并以连续不断的非平凡游戏玩法为属性，即「无限谜题」。如果一切都失败了，游戏中断了，或者开发者离开了，但如果游戏仍然有需求，可分叉性则可以确保游戏持续存在。

### 3. 开放经济

`真实资产拥有权`：在全链游戏中，虚拟世界中的游戏资产可以通过区块链技术实现真实拥有权。玩家可以自由买卖、交易游戏资产，而不再受限于游戏内部的封闭性经济系统。

`唯一可验证的数字资产`：全链游戏中的游戏资产采用非同质化代币（Non-Fungible Token，NFT）进行表示，每个资产都是独一无二且具备唯一性。这使得玩家可以凭借区块链上的数据验证资产的真实性和稀缺性。

`去中心化的交易市场`：全链游戏通过智能合约和去中心化交易平台，为玩家提供了开放、公平、高效的交易环境。玩家可以直接与其他玩家进行买卖交易，消除了传统游戏中的中介和对第三方平台的依赖。
