## Get Started

### 1. Download the starter kit

`git clone https://github.com/curio-research/keystone-starter-kit.git`

### 2. Run Server

```bash
cd server
go mod tidy
go run main.go
```

### 3. Run Client

```bash
cd client
yarn
yarn start
```
